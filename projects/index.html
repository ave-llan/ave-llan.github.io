computational counterpoint

musicmachine.io is a visual composition tool which helps you compose following the rules of species counterpoint. Data is modeled with a set of modules that represent a musical grammar in the form of a graph.

grammar graph

grammar-graph takes a context-free grammar and converts it into a decision-making graph. Can produce interactive Guides which generate valid sentences from the grammar.
